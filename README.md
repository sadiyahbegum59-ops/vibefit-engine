# VibeFit Engine 👗

Fashion × AI × Personal Expression

An intelligent fashion recommendation system that suggests outfits based on mood, occasion, aesthetic preference, and style compatibility.

---

## Features

✔ Mood based recommendations  
✔ Occasion based outfit suggestions  
✔ Traditional and modern outfit support  
✔ Aesthetic compatibility scoring  
✔ Rule based recommendation engine  
✔ Interactive user interface  
✔ Fashion centered logic system  

---

## Tech Stack

- Python  
- Gradio  
- Recommendation Engine  
- Rule Based Logic  

---

## Project Structure

```text
app.py → Handles user interface

data.py → Stores outfit database

recommendation.py → Outfit recommendation logic

scoring.py → Compatibility scoring

utils.py → Helper functions
```
